# Registry components

### Folder Structure

ref: https://ui.shadcn.com/docs/registry/registry-item-json

- blocks: Use for complex components with multiple files.
- components: Use for simple components.
- lib: Use for utility functions and helpers.
- hooks: Use for custom React hooks.
- ui: Use for UI components and single-file primitives.
- styles: Use for global styles and theming.
- page: Use for page or file-based routes.
